WAVE Web Accessibility Report

I made several key accessibility changes to ensure the website is usable for everyone. First, I wrapped all the main content in a <main> I also converted images to AVIF format for file size and added explicit width and height attributes to the server image, preventing unexpected layout shifts that can be disorienting for users with vestibular disorders or those using assistive technologies. Additionally, I ensured all images have descriptive alt text and that the page maintains proper semantic structure with heading hierarchy. These changes may seem subtle, but they make a significant difference for users who rely on assistive technologies to browse the web, ensuring my portfolio is inclusive and accessible to all visitors.
