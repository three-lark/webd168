SASS DEMO SETUP INSTRUCTIONS

1. Install Node.js from https://nodejs.org
2. Open this folder (sass) in Visual Studio Code.
3. Open a terminal inside VS Code.
4. Run: npm install
   (This installs Sass, PostCSS, and Autoprefixer locally.)
5. To start the Sass compiler in terminal, run:
   npm run sass
6. To preview the page, install the Live Server extension and right-click index.html ‚Üí "Open with Live Server".
